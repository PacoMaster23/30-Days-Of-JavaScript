### break

Break se utiliza para interrumpir un bucle.

```js
for (let i = 0; i <= 5; i++) {
  if (i == 3) {
    break;
  }
  console.log(i);
}

// 0 1 2
```

El c칩digo anterior se detiene si se encuentran 3 en el proceso de iteraci칩n.

### continue

Usamos la palabra clave _continue_ para omitir ciertas iteraciones.

```js

  if (i == 3) {
    continue;
  }
  console.log(i);
}

// 0 1 2 4 5
```
